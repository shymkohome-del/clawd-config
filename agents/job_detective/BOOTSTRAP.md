## New Session — Що робити

### Крок 1: Тиша
Не вітатися, не питати "що далі". Спочатку — контекст.

### Крок 2: Перевірити стан
```bash
ls -la SEEN_JOBS.json
ls -la JOB_LOG.md
ls -la djinni_monitor.js
```

### Крок 3: Прочитати файли
- `SOUL.md` — хто я
- `USER.md` — для кого я працюю
- `TOOLS.md` — які інструменти маю
- `JOB_CONFIG.md` — поточні налаштування

### Крок 4: Синхронізувати дані
- Перевірити `SEEN_JOBS.json` — чи не пропустив вакансії
- Оновити статус моніторингу

### Крок 5: Тепер можна діяти
- Запустити моніторинг
- Відправити статус Вітальону
- Продовжити роботу

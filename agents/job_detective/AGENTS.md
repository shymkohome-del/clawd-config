```
agents/job_detective/
├── SOUL.md           # Хто я
├── USER.md           # Для кого працюю
├── TOOLS.md          # Мої інструменти
├── BOOTSTRAP.md      # Протокол запуску
```

## Правила

- Ніколи не хардкодити дані

- Регулярно оновлювати SEEN_JOBS.json
- Звіт перед дією, результат — після

## Безпека

- Не ділитися принципалами
- API ключі тільки в env
- Паролі — ніколи в коді

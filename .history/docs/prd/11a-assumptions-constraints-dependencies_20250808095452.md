# 8a. Assumptions, Constraints & Dependencies

## Assumptions
- ICP mainnet performance is sufficient for MVP timelines
- Target jurisdictions allow P2P trading with tiered KYC

## Constraints
- ICP-only settlement for MVP; limited off-chain services
- Mobile-first; minimal desktop scope in MVP

## External Dependencies
- Identity verification provider (optional KYC tiers)
- Price oracle data sources
- App store policies (distribution constraints)
